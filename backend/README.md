# Backend – Freshdesk HubSpot Portal

This is the backend service for the Freshdesk–HubSpot Integration Portal.  
It provides authentication, integrations with Freshdesk and HubSpot, and webhook handling.

---

## Tech Stack

- Node.js
- Express
- MongoDB
- JWT Authentication
- Axios

---

## Features

- User authentication (signup/login)
- Freshdesk API integration
- HubSpot CRM integration
- Freshdesk webhook ingestion
- Webhook event logging

---

## Setup Instructions

### 1. Install dependencies

```bash
npm install
```
